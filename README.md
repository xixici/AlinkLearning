## 目的

为了学习Alink以及所涉及的Flink知识，本仓库有两大目的

首先，旨在使用scala实现Alink文档中提到的所有特性，包括但不仅限于以下:

- [x] 分类算法
    - 随机森林分类
    - 支持向量机
    - 决策树
    - 朴素贝叶斯方法
    - 多层感知机
    - gbdt二分类
    - softmax多分类
    - 逻辑回归
    - OneVsRest
- [x] 回归算法
    - gbdt回归
    - 线性回归
    - 随机森林回归
    - 保序回归
    - 广义线性回归
    - AFT生存回归
    - 决策树回归
    - Lasso回归
    - 岭回归
- [x] 聚类算法
    - 二分K均值聚类
    - KMeans
    - 高斯混合模型
    - LDA
- [x] 协同过滤
    - ALS
- [x] 异常检测
    - SOS
- [x] 关联规则
    - FP-Growth
    - PrefixSpan
- [x] 在线学习
    - ftrl  
- [ ] 特征工程
- [ ] 模型的选择和调参
- [x] 模型的保存和加载
